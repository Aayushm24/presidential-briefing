# Aayush's AI commentary posts — few-shot examples

These are the DIRECT style reference when writing about AI tools, startups, and products.
NOT "read for inspiration" — use these as structural templates.
Fill the structure with today's news. Do not change the structure.

Key patterns across all 4:
1. Lowercase opening statement (observation, not setup)
2. "At Atlan, we..." appears naturally mid-post, not as an intro
3. Contrast labels: "That's distribution." "That's the third." after named examples
4. Recap tags: "None of that is a button." "That war is over."
5. Doubt/IMO markers as genuine opinion, not decoration
6. p.s. reveals the personal stake — honest, not humble-braggy
