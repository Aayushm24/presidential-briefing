# creatives/

**This folder only holds reusable reference material.** Each day's creatives now live under `workspace/YYYY-MM-DD/creatives/` alongside that day's brief, posts, and scan summary.

## What's here

- `examples/` — hand-curated reference examples for designers and the `/design-council` skill. Not tied to a specific date.

## Where are today's creatives?

`workspace/YYYY-MM-DD/creatives/` — e.g. `workspace/2026-04-24/creatives/option-1-receipt.png`.

## Why this changed (2026-04-24)

Keeping creatives next to the post they support (`workspace/YYYY-MM-DD/posts.json`) means one directory per day, easier to zip and hand off, and matches how `brief.md` and the scan artifacts live. The `/design-council` skill writes here; the `/publish` pipeline can pick up the PNG automatically.
